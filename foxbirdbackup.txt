Firefox and Thunderbird Profile 1.1.4.1
(Explication en Français à la fin de ce fichier)

OBJECT
Save and restore :
Mozilla Firefox  profiles.
Thunderbird profiles

INSTALLATION
Windows: Extract archive and run InstallFireback.exe, then follow instructions of the setup program.
Installer install the proper version regarding 32 or 64 bit OS.
Linux: Extract linux archive in a folder.

USAGE
The first panel, Backup creation, lists current user profiles.
User can backup or modify a profile.

The second panel, Restoration and backup management, lists backups previously done
or imported. User can restore a backup in the profile location. User can export a backup on a different folder
or drive. He can import an external backup archive.
Imported archives are then listed in the second panel. Modifications include
change of profile name and default activation of a profile.

UNINSTALLATION
Windows: Use control panel's program management system to uninstall the program.
Linux: Delete files extracted on install.

NOTICE
Fireback software can be used free of charge for a non commercial use.


Use Fireback at your own risks. Profiles manipulation can corrupt or lose critical data.

Sauvegarde de profil Firefox  et Thunderbird

OBJET
Sauvegarde et restaure les profils
Mozilla Firefox
Thunderbird

INSTALLATION
Windows: Extraire l'archive et exécuter InstallFireback.exe, puis suivre les instructions
du programme d'installation.
L'installateur installe la version correspondant au SE, 32 ou 64 bits.
Linux: Extraire l'archive Linux dans un répertoire.

UTILISATION
Le premier onglet, Création de sauvegardes, liste les profils de
l'utilisateur courant. L'utilisateur peut sauvegarder un profil.

Le second onglet, Restauration et gestion de sauvegardes, liste les sauvegardes
précédemment effectuées ou importées. L'utilisateur peut restaurer une sauvegarde
a l'emplacement du profil actuel ou a un autre emplacement. Si l'utilisateur choisit
de ne pas écraser le profil actuel, le progamme crée et active un novueau profil.
L'utilisateur peut exporter une sauvegarde archivée dans un dossier ou un disque différent.
Il peut importer une sauvegarde archivée externe. Les archives importée sont ensuite listées
dans le second onglet.

DESINSTALLATION
Windows: Utiliser le système de gestion de programmes du panneau de configuration pour
désinstaller le programme.
Linux: Supprimer les fichiers extraits lors de l'installation.

NOTICE
Le logiciel FoxBirdBackup peut être utilisé gratuitement pour une utilisation non commerciale.


Utilisez FoxBirdBackup à vos risques et périls. La manipulation des profils peut corrompre
ou détruire des données critiques.
