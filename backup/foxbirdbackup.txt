Firefox and Thunderbird Profile Backup
FoxBirdBackup Version 1.1.0.0

(Explication en Français à la fin de ce fichier)

OBJECT
Save, restore and edit :
Mozilla Firefox 4, 5, 6 and 7 profiles.
Thunderbird 5 profiles

INSTALLATION
Extract archive and run InstallFireback.exe, then follow instructions of the setup program.

USAGE
The first panel, Backup creation and importation, lists current user profiles.
User can backup or modify a profile and import an external backup archive.
Imported archives are then listed in the second panel. Modifications include
change of profile name and default activation of a profile.

The second panel, Restoration and backup management, lists backups previously done
or imported. User can restore a backup in the profile location or in another
location. If the user choose to not overwerite the current profile, the program
creates and activate a new profile. User can export a backup on a different folder
or drive.

UNINSTALLATION
Use control panel's program management system to uninstall the program.

FoxBirdBackup Version 1.0.0.8

OBJET
Sauvegarde, restaure et édite les profils
Mozilla Firefox 4, 5, 6 et 7
Thunderbird 5

INSTALLATION
Extraire l'archive et exécuter InstallFireback.exe, puis suivre les instructions
du programme d'installation.

UTILISATION
Le premier onglet, Création et importation de sauvegardes, liste les profils de
l'utilisateur courant. L'utilisateur peut sauvegarder ou modifier un profil, et
importer une sauvegarde archivée externe. Les archives importée sont ensuite listées
dans le second onglet. Les modifications comprennent le changement du nom du profil
et l'activation par défaut d'un profil.

Le second onglet, Restauration et exportation de sauvegardes, liste les sauvegardes
précédemment effectuées ou importées. L'utilisateur peut restaurer une sauvegarde
a l'emplacement du profil actuel ou a un autre emplacement. Si l'utilisateur choisit
de ne pas écraser le profil actuel, le progamme crée et active un novueau profil.
L'utilisateur peut exporter une sauvegarde archivée dans un dossier ou un disque différent.

DESINSTALLATION
Utiliser le système de gestion de programmes du panneau de configuration pour
désinstaller le programme.
